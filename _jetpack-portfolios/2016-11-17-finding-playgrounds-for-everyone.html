At NPR, we wanted to build an application that helped parents nationwide find playgrounds with features that make them more accessible to kids with disabilities. The problem? No such database exists from an official source. So I helped our team collect, request and scrape data from various sources. We later added a feature allowing parents to add their own locations to our incomplete database, helping improve a useful tool.
